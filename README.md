# Bragibytes

A personal blog about building software by directing AI models and agents.

**Live site**: [bragibytes.com](https://bragibytes.com)

## Tech Stack

- Astro
- MDX + Content Collections
- Tailwind CSS
- Shiki (syntax highlighting)
- Pagefind (static search)
- TypeScript

The site is deliberately small and lightweight.

## Local Development

```bash
npm install
npm run dev
```

## Writing

See the `scripts/new-post.ts` helper or just create a new file in `src/content/blog/`.
